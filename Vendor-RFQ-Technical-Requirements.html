Request for Quotation (RFQ)

Professional PA System for Ministry Event Venue

Project Information

Issuing Organization: Syria Ministry of Interior Project: Professional Audio System Installation RFQ Issue Date: January 2025 Proposal Deadline: [To be specified] Project Location: [Venue to be specified], Syria

Project Overview

The Syria Ministry of Interior seeks quotations for a complete professional public address (PA) system for a ministry event venue. The system must provide high-quality audio reinforcement for government events, conferences, and official gatherings with capacity of up to 2000 attendees.

Technical Requirements

1. WIRELESS MICROPHONE SYSTEM

Quantity: 4-6 wireless microphone channels Technical Specifications:
  • Digital wireless technology (preferred) or high-quality analog
  • UHF frequency range compatible with Syrian telecommunications regulations
  • CRITICAL: All wireless equipment must operate on frequencies approved by Syrian Telecommunications & Post Regulatory Authority (SY-TPRA)
  • Operating range: Minimum 100 meters line-of-sight
  • Dual capability: Each channel must support BOTH handheld transmitter AND bodypack transmitter with lavalier microphone (interchangeable)
  • Rechargeable battery system preferred (8+ hour runtime)
  • Rack-mountable receivers
  • Low latency: <5ms preferred
  • RF stability with frequency coordination capability
  • Minimum 20 Hz - 20 kHz frequency response
  • Signal-to-noise ratio: >100 dB preferred
What Vendor Must Provide:
  • Number of receiver units (specify single/dual/quad channel)
  • Number of handheld transmitters
  • Number of bodypack transmitters
  • Number of lavalier microphones
  • Battery type and quantity (rechargeable system preferred)
  • Charging solution
  • Antenna system (if required)
  • Specify exact frequency band/range of proposed equipment

2. WIRED MICROPHONE SYSTEM (OPTIONAL ALTERNATIVE)

Quantity: 2 wired microphones Technical Specifications:
  • Dynamic cardioid vocal microphones
  • Professional grade, suitable for government/ministry use
  • Frequency response optimized for speech
  • Built-in shock mount
  • On/off switch optional
What Vendor Must Provide:
  • Microphone specifications
  • Microphone stands (adjustable boom stands)
  • XLR cables (minimum 15m length each)

3. IN-EAR MONITOR SYSTEM

Quantity: 2 complete in-ear monitor systems Technical Specifications:
  • Wireless personal monitoring system
  • UHF frequency range (must comply with Syrian regulations)
  • Operating range: Minimum 100 meters
  • Stereo operation
  • Bodypack receiver with belt clip
  • Professional in-ear headphones/earpieces
  • Rechargeable battery system (8+ hour runtime)
  • Low noise floor
  • Frequency response: 25 Hz - 15 kHz minimum
What Vendor Must Provide:
  • Transmitter units (rack-mountable)
  • Bodypack receivers (quantity: 2)
  • In-ear monitor earpieces (quantity: 2 pairs)
  • Battery system and chargers
  • Antenna system (if required)

4. LINE ARRAY SPEAKER SYSTEM

Quantity: Main PA system for 1000-person venue Technical Specifications: Main Arrays:
  • Lightweight line array elements suitable for truss mounting
  • Maximum weight per element: 15 kg
  • Quantity: 12 elements total (3 arrays of 4 elements, or 2 arrays of 6 elements each, left/right)
  • Powered/active speakers preferred
  • Minimum SPL: 125 dB @ 1m per element
  • Frequency response: 60 Hz - 20 kHz minimum
  • Horizontal coverage: 80-100° per element
  • Vertical coverage: 8-12° per element
  • Flying hardware included (rigging frames, safety cables)
  • Compatible with 290-300mm box truss
Subwoofers:
  • Powered/active subwoofers
  • Quantity: 4 units
  • Frequency response: 35-150 Hz
  • Minimum SPL: 130 dB @ 1m
  • 18" driver preferred
  • Ground-stack or flyable design
System Requirements:
  • All speakers must support 220V/50Hz power (Syria standard)
  • Built-in DSP for system tuning preferred
  • Network control capability preferred but not required
  • Venue coverage: 60m depth × 50m width
  • Even SPL distribution throughout coverage area
What Vendor Must Provide:
  • Detailed specifications for proposed line array elements
  • Detailed specifications for subwoofers
  • Flying frames and rigging hardware
  • Safety cables and attachment hardware
  • Speaker cables (specify length and type)
  • Power cables
  • System tuning/alignment plan

5. MIXING CONSOLE (OPTIONAL - If Not Already Owned)

Quantity: 1 mixing console Technical Specifications:
  • Minimum 16 input channels
  • Digital console preferred
  • Minimum 6 auxiliary sends (for monitor mixes)
  • Built-in effects processor
  • Parametric EQ on all channels
  • USB recording interface preferred
  • Motorized faders preferred but not required
  • Scene recall capability preferred
What Vendor Must Provide:
  • Console specifications
  • Control surface if digital
  • Required cables and connections
  • Tablet/computer control option (if available)

6. CABLING AND ACCESSORIES

What Vendor Must Provide:
  • XLR cables: Quantity and lengths (stage to FOH position, approximately 30-50m runs)
  • Power distribution system
  • Equipment rack (12U minimum for wireless/IEM equipment)
  • Cable management accessories
  • Microphone stands for wired mics (if applicable)
  • All necessary adapters and connectors

Venue Specifications

Venue Dimensions:
  • Depth: 60 meters
  • Width: 50 meters
  • Ceiling Height: 5-8 meters (for rigging)
  • Capacity: 2000 people seated
Available Infrastructure:
  • Power: 220V/50Hz (Syria standard)
  • Available Circuits: 2 × 20A dedicated circuits
  • Truss: 290-300mm box truss available for speaker rigging
  • FOH Position: 15-20m from stage

Power Requirements

System Must Operate On:
  • Voltage: 220V/50Hz (Syria standard)
  • Maximum power draw: 17 kW total system
  • Vendors must specify:

- Power consumption per component

- Total system power requirements

- Recommended circuit configuration

Regulatory Compliance

CRITICAL REQUIREMENT: Wireless Frequency Compliance

All wireless equipment (microphones and in-ear monitors) MUST:
  1. Operate on frequencies approved by Syrian Telecommunications & Post Regulatory Authority (SY-TPRA)
  2. Common legal bands in Syria: 518-566 MHz, 606-614 MHz (must be verified)
  3. Vendor must provide documentation proving frequency legality in Syria
  4. Equipment must be ordered with correct frequency band for Syria
Vendor Responsibilities:
  • Confirm proposed equipment can be supplied in Syria-legal frequencies
  • Provide frequency coordination plan for all wireless channels
  • Provide documentation for Syrian customs clearance
  • Ensure compliance with Syrian telecommunications law
Ministry Responsibility:
  • Obtain official frequency authorization from SY-TPRA before equipment order
  • Provide copy of frequency authorization to selected vendor
